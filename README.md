# proxy ftp

Proxy ftp pour un devoir de cours de la ressource R3.05 

## utilisation

Lancer le proxy

```bash
make
./proxy
```

Se connecter au proxy

```bash
ftp 127.0.0.1 <port fournis par le proxy>
```
